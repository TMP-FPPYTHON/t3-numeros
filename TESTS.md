# Ejercicios Ciclos For!

## Configuración de pruebas con **pytest**

`nota:` para todo las pruebas puedes usar esta configuración:
### Setup command (optional)
```
pip3 install pytest
```

- ## 00 Suma Pares
    ### Run command
    ```
    pytest assignments/00SumaPares
    ```
- ## 01 Promedio
    ### Run command
    ```
    pytest assignments/01Promedio
    ```
